# Optimization-and-Analysis-of-Graph-Algorithms

## Опис проєкту
Даний програмний продукт розроблений мовою C++ у межах курсової роботи.  
Застосунок дозволяє користувачу задати початкову та кінцеву точки маршруту, після чого програма обчислює найкоротший шлях між ними та відображає орієнтовний час пересування.
Проєкт демонструє роботу з алгоритмами графів, логікою маршрутизації та структурою програмного коду мовою C++.

## Функціонал йогго можливості
- Введення початкової та кінцевої точки маршруту  
- Побудова шляху  
- Розрахунок довжини маршруту  
- Обчислення орієнтовного часу руху
