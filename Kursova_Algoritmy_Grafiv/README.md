# Оптимізація та аналіз алгоритмів роботи з графами

# Route Planner (C++)
Десктопний застосунок, розроблений мовою **C++**, для побудови найкоротшого маршруту між двома точками та розрахунку орієнтовного часу в дорозі.

## Технологічний стек
- Мова програмування: **C++**
- Алгоритми роботи з графами (пошук найкоротшого шляху)

## Мета проєкту
Розробка простого та зрозумілого застосунку, який дозволяє користувачу знайти оптимальний маршрут між двома точками та отримати інформацію про довжину і час пересування.

## Основні можливості
- введення початкової та кінцевої точки маршруту;
- побудова шляху;
- розрахунок довжини маршруту;
- обчислення орієнтовного часу руху:
   
## Примітки
Проєкт підготовлений з урахуванням можливості подальшого розширення, зокрема додавання графічного інтерфейсу та нових функцій.
